# crypto-experts

- This is the result of an intensive 12 weeks full-stack-web development bootcamp by Spiced Academy.
- A single-page application website allowing users to join a social network of crypto enthusiasts.
- On registration every user receives certain amount of FIAT play money that he can use to trade with cryptocurrencies.
- The goal is to gain experience with the crypto market and take a good place in the ranking of best traders.
- The CoinGecko API powers the latest market data and the visualisations of the market history are done with Chart.js.
