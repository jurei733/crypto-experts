# crypto-experts

- This is the result of an intensive 12 weeks full-stack-web development bootcamp by [SpicedAcademy].
- A single-page application built with [React]/[Redux] allowing users to join a social network of crypto enthusiasts.
- On registration every user receives certain amount of FIAT play money that he can use to trade with cryptocurrencies.
- The goal is to gain experience with the crypto market and take a good place in the ranking of best traders.
- The [CoinGecko] API powers the latest market data and the visualisations of the market history and user's portfolio are done with [Chart.js].
- Its possible to add a biography, send/receive/decline friend requests, visit other usersâ€™ profile and chat online.
